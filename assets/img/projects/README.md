Curated project visuals
=======================

This folder contains only public-facing images extracted from local notebook outputs after review or downloaded from public free-use sources for portfolio cards.

Rules:
- Do not copy raw notebooks, CSV files, datasets, credentials, or private bootcamp folders here.
- Do not publish screenshots with local paths, personal information, tokens, or private notes.
- Avoid dataset images with identifiable people unless the image is explicitly public and approved for portfolio use.

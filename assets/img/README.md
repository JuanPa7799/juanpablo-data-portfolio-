# Imagenes del portafolio

Los assets actuales fueron generados localmente con imagegen para evitar hotlinks y dependencias externas. Se usan como sustitutos seguros de imagenes tipo Pexels mientras se decide si se reemplazan por fotos especificas.

## Assets

- `hero-analytics.png`: hero principal de analitica y dashboards.
- `data-analyst.png`: ruta Data Analyst, KPIs y business analytics.
- `data-scientist.png`: ruta Data Scientist, modelos y evaluacion.
- `machine-learning.png`: ruta Machine Learning, pipeline aplicado.
- `projects/`: visuales curados extraidos de notebooks locales o recreados de forma segura.

## Foto de perfil

LinkedIn no permitio extraer automaticamente la foto publica sin pasar por login/cookies. La Home usa un avatar de iniciales como fallback. Cuando haya una foto aprobada, colocar la fuente fuera de Git en `assets-source/profile.jpg` y copiar solo la version final publica como `profile.jpg`.

## Nota de licencia

La intencion visual toma como referencia sitios con imagenes libres tipo Pexels, pero no se copian archivos de Pexels porque la descarga automatizada fue bloqueada. Si se reemplazan despues, usar imagenes descargadas localmente y documentar fuente/licencia aqui.

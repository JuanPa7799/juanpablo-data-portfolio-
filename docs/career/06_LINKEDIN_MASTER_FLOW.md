# Flujo LinkedIn -> Portafolio -> Dashboards por rol

## Objetivo

Que cualquier persona que llegue desde LinkedIn pueda elegir rapidamente la ruta que mas le interesa: Data Analyst Jr, Data Scientist Jr o Machine Learning Jr.

## Link unico recomendado

LinkedIn debe apuntar a:

```text
https://JuanPa7799.github.io/juanpablo-data-portfolio-/
```

Ese link no debe mandar directo a un proyecto aislado. Debe mandar al dashboard principal.

## Headline de LinkedIn recomendado

Data Scientist Jr | Machine Learning | Python | SQL | Modelado Predictivo | Ingenieria Mecatronica

Alternativas segun vacante:

- Data Analyst Jr | SQL | Python | Dashboards | KPIs | Business Analytics
- Data Scientist Jr | Python | ML | Estadistica | SQL | Predictive Modeling
- Machine Learning Jr | Python | TensorFlow | Scikit-learn | Pipelines | Applied AI

## Seccion Acerca de

Debe contar una historia breve:

1. formacion en ingenieria;
2. transicion a datos;
3. proyectos con metricas;
4. interes por roles junior;
5. link al portafolio.

## Experiencia y educacion

- TripleTen debe aparecer como formacion, certificacion o programa academico, no como empleo tradicional.
- La experiencia de ingenieria debe conectar con datos, automatizacion, analisis, senales o sistemas.
- Evitar bullets vacios, errores de encoding y tecnologias duplicadas.

## Proyectos destacados en LinkedIn

1. Clasificacion de perturbaciones electricas.
2. Prediccion de churn Interconnect.
3. OilyGiant Risk & Profit.
4. Sweet Lift Taxi Forecasting.
5. Good Seed Age Vision.

Cada proyecto debe tener:

- titulo claro;
- problema resuelto;
- metricas;
- stack;
- link al portafolio o GitHub curado.

## Flujo ideal del reclutador

```text
LinkedIn
  -> dashboard principal
    -> Data Analyst / Data Scientist / Machine Learning
      -> proyecto relevante
        -> CV del rol ES/EN
        -> contacto
```

## Checklist antes de actualizar LinkedIn

- Dashboard principal listo.
- Tres dashboards por rol listos.
- CVs por rol aprobados.
- Links publicos probados.
- Proyectos sin datos sensibles.
- Version ES/EN disponible en las paginas principales.

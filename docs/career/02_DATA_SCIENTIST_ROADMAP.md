# Ruta Data Scientist Jr

## Posicionamiento

**Titulo objetivo:** Data Scientist Jr | Python | Machine Learning | Modelado Predictivo | Estadistica | SQL

Esta ruta debe mostrar flujo completo: problema, datos, EDA, feature engineering, baseline, modelos, validacion, metricas, interpretacion y limitaciones.

## Dashboard del rol

El dashboard de Data Scientist debe priorizar:

- modelos comparados;
- metricas por problema;
- decisiones de validacion;
- feature engineering;
- limitaciones y siguientes pasos.

## Roadmap de habilidades

- Python: pandas, numpy, sklearn, notebooks limpios.
- Estadistica: hipotesis, distribuciones, intervalos, validacion.
- ML: regresion, clasificacion, ensembles, metricas.
- Feature engineering: variables temporales, codificacion, imputacion.
- Interpretabilidad: feature importance, errores, limitaciones.
- Comunicacion: resumen ejecutivo y lectura tecnica.

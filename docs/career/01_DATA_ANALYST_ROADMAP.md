# Ruta Data Analyst Jr

## Posicionamiento

**Titulo objetivo:** Data Analyst Jr | SQL | Power BI | Python | KPIs | Storytelling con datos

Esta ruta debe comunicar capacidad para transformar datos en decisiones de negocio: dashboards, KPIs, SQL, visualizaciones claras y recomendaciones accionables.

## Dashboard del rol

El dashboard de Data Analyst debe priorizar:

- KPIs de negocio;
- filtros visuales;
- storytelling ejecutivo;
- proyectos con impacto comercial;
- botones a dashboard, reporte y codigo.

### Proyectos actuales que pueden entrar

| Proyecto | Estado | Por que sirve |
|---|---|---|
| OilyGiant Risk & Profit | Listo para pagina curada | Muestra decision de negocio con riesgo y beneficio |
| Rusty Bargain Pricing | Listo para pagina curada | Conecta modelo con producto/app y velocidad |
| Interconnect Churn | Listo para pagina curada | Caso de retencion y segmentacion de clientes |
| Megaline Plan Recommendation | Listo como proyecto ML/analitica | Recomendacion de planes con comportamiento de usuarios |

### Proyectos faltantes recomendados

1. Dashboard BI de KPIs comerciales.
2. Analisis SQL con consultas avanzadas.
3. Automatizacion de reporte semanal en Python.

## Roadmap de habilidades

- SQL: JOINs, CTEs, window functions, agregaciones por cohortes.
- BI: Power BI o Tableau con filtros, medidas y KPIs.
- Python: pandas, limpieza, visualizaciones, automatizacion.
- Negocio: retencion, ventas, conversion, margen, riesgo.
- Comunicacion: resumen ejecutivo, insights y recomendaciones.

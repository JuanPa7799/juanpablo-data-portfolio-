# Dashboard Machine Learning Jr

## Proposito

Mostrar que tienes base tecnica para construir modelos y avanzar hacia sistemas reutilizables: pipelines, validacion, estructura, APIs y buenas practicas.

## URL futura

`dashboards/machine-learning/`

## Estructura propuesta

1. Hero con enfoque ML aplicado a ingenieria.
2. Panel de capacidades:
   - senales;
   - vision;
   - NLP;
   - forecasting;
   - pipelines.
3. Proyecto estrella:
   - Calidad de energia.
4. Cards de proyectos:
   - Good Seed Age Vision;
   - IMDB Sentiment NLP;
   - Rusty Bargain Pricing;
   - Sweet Lift Taxi.
5. Roadmap tecnico visible:
   - notebook limpio;
   - script modular;
   - pipeline;
   - API;
   - Docker;
   - monitoreo.

## Interacciones

- Filtro por dominio: senales, vision, NLP, forecasting, tabular.
- Mini diagrama de pipeline.
- Comparador calidad vs velocidad.
- Indicador de estado: notebook, script, API, demo.
- Boton ES/EN visible.

## Copy base

Mi enfoque de Machine Learning combina modelos aplicados, criterio de ingenieria y pasos concretos hacia soluciones reutilizables. Trabajo para que cada proyecto pase de resultado experimental a flujo reproducible.

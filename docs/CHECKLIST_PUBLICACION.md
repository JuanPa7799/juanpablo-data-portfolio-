# Checklist de publicacion

Antes de activar o actualizar GitHub Pages:

- [ ] Renombrar el repositorio remoto a `juanpablo-data-portfolio`.
- [ ] Confirmar que GitHub Pages publique desde `main` y `/root`.
- [ ] Revisar que no haya archivos de variables de entorno, tokens, API keys ni credenciales.
- [ ] Revisar que no haya rutas locales absolutas en archivos publicados.
- [ ] Revisar que no se hayan publicado datasets completos privados.
- [ ] Probar `index.html` en escritorio y movil.
- [ ] Probar paginas en `projects/calidad-energia/` y `projects/interconnect-churn/`.
- [ ] Probar descarga de `assets/pdf/Portafolio_de_evidencias_V2.pdf`.
- [ ] Probar links externos a GitHub, LinkedIn y email.
